public class Table<Key, Element> where Key : Hashable

extension Table: CustomStringConvertible, CustomDebugStringConvertible
where Key: CustomStringConvertible, Element: CustomStringConvertible

Table is a 2-Dimensional dictionary. It consist of rows and columns. In a table, every value is stored using two keys, its row and column.
