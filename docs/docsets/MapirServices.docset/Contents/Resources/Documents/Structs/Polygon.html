Polygon

 
public struct Polygon

A Polygon object represents a closed shape with 4 or more vertices specified as CLLocationCoordinate2D objects. Polygons are used to demonstrate a building, island, lake or any other area that you want to highlight.

The vertices are connected in the order that you provide. You should close the polygon by specifying same CLLocationCoordinate2Ds as the first and the last vertex.
